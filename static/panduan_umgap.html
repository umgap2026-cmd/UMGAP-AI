Akun & Login

Cara masuk, daftar, dan reset password

👥 Semua Pengguna
🔑

Login ke UMGAP

Gunakan email & password terdaftar
  • 1
    Buka browser → akses umgap-ai.onrender.com
  • 2
    Masukkan Email dan Password
  • 3
    Klik tombol Masuk
  • 4
    Sistem mengarahkan ke dashboard sesuai role secara otomatis
💡 Tersedia login dengan akun Google untuk opsi lebih cepat.
📝

Daftar Akun Baru

Akun baru otomatis mendapat role Karyawan
  • 1
    Klik Daftar di halaman login
  • 2
    Isi Nama Lengkap, Email, dan Password (min. 6 karakter)
  • 3
    Klik Daftar — langsung masuk otomatis
⚠️ Akun baru mendapat role Karyawan. Untuk role Admin, minta admin mengubahnya di halaman Kelola User.
🔒

Lupa Password

Reset via kode OTP ke email
  • 1
    Klik Lupa Password di halaman login
  • 2
    Masukkan Email terdaftar → klik Kirim OTP
  • 3
    Buka email, cari kode OTP 6 digit dari UMGAP
  • 4
    Masukkan OTP + password baru → klik Reset
⏰ Kode OTP hanya berlaku 10 menit. Jika kadaluarsa, ulangi dari awal.

Absensi Harian

Cara submit kehadiran setiap hari

👤 Karyawan
📍

Cara Submit Absensi

Butuh GPS aktif dan kamera
  • 1
    Buka menu Absensi dari dashboard
  • 2
    Pilih Tipe Kehadiran yang sesuai
  • 3
    Isi Catatan jika perlu (opsional)
  • 4
    Tap Ambil Lokasi GPS — tunggu koordinat muncul ✓
  • 5
    Tap area kamera → ambil Foto Selfie menggunakan kamera depan
  • 6
    Semua syarat hijau ✓ → tap Kirim Absensi
✅ Setelah berhasil, otomatis berpindah ke tab Riwayat untuk konfirmasi.
📋

Tipe Kehadiran

Pilih sesuai kondisi hari ini
TipeKapan Digunakan
HadirMasuk tepat waktu
TerlambatDatang melebihi jam kerja
SakitTidak masuk karena sakit
IzinTidak masuk dengan izin
Tidak HadirAbsen tanpa keterangan
📈

Riwayat & Estimasi Gaji

Tab Riwayat di halaman Absensi
  • Rekap bulan ini: Hadir, Sakit, Izin, Absen, Telat
  • Estimasi gaji berdasarkan kehadiran
  • Detail absensi per hari
ℹ️ Estimasi gaji bersifat perkiraan. Gaji resmi ditetapkan admin melalui modul Payroll.

Input Laporan Penjualan

Perlu persetujuan admin

AI Caption Generator

Generate caption media sosial otomatis

👥 Semua Pengguna
✨

Cara Generate Caption

Menghasilkan 3 variasi siap pakai
  • 1
    Buka menu AI Caption
  • 2
    Isi Nama Produk (wajib)
  • 3
    Isi Harga dan Brand (opsional)
  • 4
    Pilih Platform: Instagram / TikTok / WhatsApp
  • 5
    Pilih Gaya: Santai / Promo / Storytelling / Serius
  • 6
    Isi Catatan Pendukung (COD, bonus, jam buka, dll.)
  • 7
    Klik Generate Caption → tunggu beberapa detik
  • 8
    Pilih variasi favorit → klik Copy Semua
💡 Makin lengkap info yang diisi, makin natural hasilnya. Gunakan Regenerate untuk variasi baru.

Kelola Absensi (Admin)

Tiga sub-menu absensi untuk admin

🛡️ Admin
👥

Absenkan Karyawan

Untuk karyawan yang tidak punya HP
  • 1
    Buka Absensi → Absen Karyawan
  • 2
    Pilih nama karyawan dari dropdown
  • 3
    Pilih Tipe Kehadiran
  • 4
    Atur Jam Kehadiran manual (opsional)
  • 5
    Tambahkan catatan → klik Submit Absensi
📋

Riwayat Absensi Semua Karyawan

Lengkap dengan foto & lokasi
  • Cari berdasarkan nama karyawan
  • Filter tipe: Ontime, Telat, Sakit, Izin, Absen
  • Tap Lihat Foto untuk foto selfie karyawan
  • Tap Lihat Lokasi untuk buka di Google Maps
✅

Persetujuan Absensi

Approve atau reject request karyawan
  • 1
    Buka Persetujuan Absensi
  • 2
    Lihat daftar request berstatus PENDING
  • 3
    Tambahkan catatan admin (opsional)
  • 4
    Klik Setujui atau Tolak

Payroll & Statistik

Hitung gaji dan pantau performa karyawan

🛡️ Admin
💰

Hitung Payroll Bulanan

Menu: Admin → Payroll

Sistem otomatis menghitung gaji berdasarkan:

  • Jumlah hari hadir bulan tersebut
  • Setting gaji harian atau bulanan per karyawan
  • Jumlah hari kerja efektif (Senin–Sabtu, Minggu libur)
ℹ️ Setting gaji diatur di Kelola User → Edit User.
📤

Export Laporan Excel

2 sheet: Absensi + Rekap Gaji
  • 1
    Buka Admin → Export
  • 2
    Pilih Tanggal Mulai dan Tanggal Selesai
  • 3
    Klik Download Excel → file .xlsx terunduh otomatis

FAQ & Troubleshooting

Masalah yang sering ditemui & solusinya

❓ GPS tidak terdeteksi saat absensi

  • Pastikan GPS / Lokasi aktif di perangkat
  • Izinkan akses lokasi untuk situs ini di pengaturan browser
  • Coba di dekat jendela atau di luar ruangan
  • Tap Perbarui Lokasi untuk coba ulang

❓ Kamera tidak bisa dibuka untuk selfie

  • Periksa izin kamera di pengaturan browser
  • Pastikan tidak ada aplikasi lain yang pakai kamera
  • Refresh halaman → izinkan kamera saat diminta

❓ Absensi tidak muncul di riwayat

  • Tarik ke bawah untuk refresh riwayat
  • Absensi masuk antrian pending sebelum diproses admin
  • Hubungi admin jika lebih dari 1 jam tidak muncul

❓ AI Caption error atau tidak keluar

  • Pastikan Nama Produk sudah diisi
  • Cek koneksi internet
  • Jika timeout, klik Generate ulang
  • Admin: pastikan Menghubungi Developer

❓ Tidak bisa login / password salah

  • Pastikan tidak ada spasi di awal/akhir email
  • Password bersifat case-sensitive
  • Gunakan Lupa Password untuk reset via OTP
  • OTP tidak datang? Cek folder Spam/Junk

❓ Laporan penjualan tidak diproses

Status PENDING menunggu admin. Hubungi admin untuk approve di menu Penjualan Admin.

❓ Karyawan tidak muncul di dropdown (Admin)

Pastikan akun karyawan tersebut role-nya diset sebagai employee di menu Kelola User.

❓ Export Excel kosong atau data tidak lengkap

  • Pastikan rentang tanggal sudah benar
  • Setting gaji harus diisi di Kelola User agar kolom gaji terisi
